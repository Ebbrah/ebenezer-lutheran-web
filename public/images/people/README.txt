People photos
============

Put passport-style photos for people here.

Naming rule (automatic)
  The website tries to load a photo automatically from:
    /images/people/<slugified-name>.jpg

Examples
  "Grace Mbata"           → public/images/people/grace-mbata.jpg
  "Mch. Grace Mbata"      → public/images/people/grace-mbata.jpg
  "Mwinj. Judith Moshi"   → public/images/people/judith-moshi.jpg

Notes
  - Use JPG (recommended). If a photo is missing, the site will show the normal placeholder circle.
  - You can still override per-person by setting `photoSrc` in the data file.
